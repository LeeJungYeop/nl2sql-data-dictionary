# NL2SQL with Data Dictionary

NL2SQL 파이프라인에 데이터 정의서를 통합하여 산업 현장의 용어 불일치 문제를 해결한 연구입니다.

## Results

| Difficulty | Baseline | + Data Dictionary |
|:----------:|:--------:|:-----------------:|
| Easy       | 85.7%    | 100.0%            |
| Medium     | 53.9%    | 69.2%             |
| Hard       | 50.0%    | 75.0%             |
| **Overall**| 62.5%    | **79.2% (+16.7%p)**|

## Tech Stack

`GPT-4o` `LangChain` `Chainlit` `MySQL` `Python`
